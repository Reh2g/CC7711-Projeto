## Objetivo:
### No simulador Webots, fazer com que um robô encontre uma caixa de madeira ‘leve’ e pare ao achar. Ao parar ele deve permanecer imóvel acender seus leds.

## Especificações do projeto:
* Robot: “e-puck”
  * controller: “my_controller”
  * supervisor: TRUE
 
* Wooden_Box
  * mass: 1
  * DEF: caixa_leve
